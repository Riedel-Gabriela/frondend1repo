Tag Link

A tag <a> faz referência a um link.

  • Utilização: <a href="https://url da página"> texto do link </a>

  • Link do Google abrindo em uma nova aba
  • Link do Google abrindo nessa mesma aba
  • Link do Google maximizando a janela
  • fotos do gatinho

Podemos atribuir, dentro do href, uma propriedade "mailto:email@emailexemplo.com" e será enviado um email para o endereço referido.

Tag Imagem

a tag <img> não necessita uma tag de fechamento. o atributo src define a fonte da imagem.

Tag Table

Sempre bom lembrar a respeito das tags de tabela:

  • <table>: indica o início da tabela
  • <th>: table header
  • <tr>: table row
  • <td>: table data
